🔗 References & Links

Curated external resources, official docs, tools, and articles organized by topic.

Cloud Cloud & Infrastructure

  • AWS Well-Architected Framework The 6 pillars of cloud architecture — operational excellence, security, reliability, performance, cost, sustainability.
  • Azure Architecture Center Guidance, blueprints, and best practices for building reliable Azure solutions.
  • Google Cloud Architecture Framework GCP design principles covering system design, operational excellence, and security.
  • CNCF (Cloud Native Computing Foundation) Landscape of cloud-native projects, SIGs, and open standards.

Architecture Architecture & Design

  • Martin Fowler — Architecture Articles and pattern catalog on software architecture, refactoring, and agile practices.
  • ThoughtWorks Tech Radar Bi-annual snapshot of technology adoption across techniques, platforms, tools, and languages.
  • ADR GitHub — Templates & Tools Architectural Decision Records: templates, tooling, and community resources.
  • C4 Model for Software Architecture Context, Container, Component, Code — a simple hierarchical approach to diagramming.

Tools Developer Tools

  • Excalidraw Free, open-source whiteboard for hand-drawn style diagrams — great for architecture sketches.
  • Mermaid.js Diagram-as-code for flowcharts, sequence diagrams, ER diagrams, and more.
  • diagrams.net (draw.io) Free diagramming tool with cloud save support, great for architecture and flow diagrams.
  • OpenAPI Initiative The specification standard for describing REST APIs. Vendor-neutral and widely adopted.

Learning Books & Learning

  • Designing Data-Intensive Applications — Kleppmann Deep-dive into data systems, distributed databases, streaming, and consistency models.
  • Building Microservices — Sam Newman The definitive guide to designing, building, and operating microservice-based systems.
  • Software Architecture: The Hard Parts — Ford, Richards, Sadalage Modern trade-off analysis for distributed architectures and decomposition decisions.
  • The Twelve-Factor App Methodology for building scalable, maintainable, cloud-ready applications.
