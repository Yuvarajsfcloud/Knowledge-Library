# Checklists

Lightweight, opinionated review checklists. Each is one page; each is meant to be **completed in a session**, not curated as a long-term document.

| Checklist | When to use |
|---|---|
| [Architecture Review](architecture-review.md) | Before pre-production go-live for a new system or major change |
| [Non-Functional Requirements (NFR)](nfr-checklist.md) | Early in design; revalidate before go-live |

For deeper compliance reviews, use the framework in [Implementation Governance (Phase G)](../../knowledge/adm/implementation-governance.md).
